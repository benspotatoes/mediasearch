# Mediasearch

Searches multiple libraries for movies and tv shows, maybe music someday.

## Installation

Add this line to your application's Gemfile:

    gem 'mediasearch'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mediasearch
